# 小丸工具箱重制版

## 项目简介
小丸工具箱重制版是基于 Python Tkinter + FFmpeg 的本地多功能媒体处理工具，覆盖视频压缩、音频处理、封装转换、反挤压、字幕封装、LUT 预览、MediaInfo 分析、批量任务与 Benchmark。

## 当前版本
- `v1.0.0525`

## 主要功能
- 视频压缩：CPU / NVIDIA / AMD / Intel(QSV) 编码器（含常用与高级模式）。
- 音频处理：音频转码、视频提取音频、倒放、左右声道复制。
- 常用工具：截取片段、分离音视频。
- 字幕工具：保留/移除轨道、封装、导出字幕。
- 反挤压页面：变形素材反挤压与画幅处理。
- 封装页面：合并、换容器、音频模式切换。
- 复古页面：老格式转码与复古参数输出。
- LUT 调色页：批量预览 LUT 缩略图。
- MediaInfo：显示媒体信息与能力指示灯（如 HDR、5.1、杜比等）。
- 任务管理：队列、并发、拖动排序、停止与日志。
- Benchmark：环境检测 + 编码器基准测试，支持编码器快速勾选（N/A/I 卡常用、全选、清除）。

## 最近已实现的关键改动
- 版本号改回手动管理，当前固定为 `v1.0.0525`。
- 设置页支持“启动时恢复上次退出页面”，多开时以最后退出的实例为准。
- 新增“完成后关机”“结束提示音”，并放在主界面进度条上方。
- 资源监控区 CPU/GPU 文本与图表对齐优化。
- 所有“输出目录/输出文件夹”按钮统一为淡蓝色样式。
- 设置窗口支持滚动条、鼠标滚轮与方向键滚动，避免滚轮误触发选项切换。
- 窗口尺寸与位置可持久化；启动时按住 Shift 可提示是否恢复默认窗口设置。
- 页面默认顺序调整为：视频、音频、常用（其余可在设置页自定义排序）。
- 页面命名“变形”改为“反挤压”。
- 视频预设区分“内置预设/用户预设”；保存进入用户预设。
- 用户预设管理（导入/导出/删除/排序）统一放到设置页面。
- 配置与预设统一迁移到：`%APPDATA%\marurebuild`。
- 输出目录兜底：无记录或路径失效时，回退 `D:\maru-output` 并同步更新界面。
- 支持导出记录 `.log`（批量超过 10 个文件时提示是否保存）。
- 新增导出文件名格式设置。
- 分辨率未选“自定义”时，自定义宽高禁用。

## 配置与数据目录
程序运行后会在以下目录保存配置和用户数据：

```text
%APPDATA%\marurebuild
```

主要包含：
- `app_settings.json`（全局设置）
- 用户预设文件（及备份）
- 运行相关配置文件

## 运行要求
- Windows 10/11
- Python 3.10+
- FFmpeg / ffprobe 可用（已加入 PATH，或放在程序可访问路径）

## 启动方式
```bash
python main.py
```
或：
```bash
python 视频高性能压缩器.py
```

## 项目结构（核心文件）
- `main.py`：入口
- `视频高性能压缩器.py`：中文入口
- `gui.py`：界面与交互逻辑
- `ffmpeg_utils.py`：FFmpeg 命令构建与调用
- `config.py`：配置常量
- `data.py`：数据结构

## 说明
- 不同硬件编码器是否可用，取决于本机 FFmpeg 编译能力与驱动支持。
- 若硬编失败，可切换 `CPU H.264 / AVC (libx264)` 进行兜底。
